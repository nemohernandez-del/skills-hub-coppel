# skills-hub-coppel
Repositorios de skills de coppel

## Este componente almacena los skills según lenguaje de programación 
Para copiar los skills según tipo de proyecto sigue los siguientes pasos:
#### 1. Clonar el repo
	  git clone https://github.com/nemohernandez-del/skills-hub-coppel.git
#### 2. Instalacion de skill ejemplo codigo Java
	  ./install.sh java 3capas-sp ../api-edocta-orquestador
